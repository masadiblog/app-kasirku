Deskripsi Aplikasi

Paket dasar App KasirKu menyediakan solusi kasir digital yang mudah digunakan untuk usaha kecil dan menengah. Fitur-fitur utama termasuk:

  • Pengelolaan stok barang
  • Pencatatan penjualan harian
  • Laporan keuangan otomatis

Khusus pengguna baru yang baru menggunakan App KasirKu, dapat memanfaatkan masa percobaan (gratis). Berlaku pada bulan pertama dan terhitung sejak akun Anda berhasil terdaftar di App KasirKu.

Semua Fitur

  • Buat akun
  • Lupa password
  • Login
  • Dashboard
    1. Produk
      • Pada bagian ini (tab produk) Anda dapat mengelola data produk dengan beberapa fitur berikut.
      • Form input produk
      • Tombol scan barcode
      • Tombol edit produk
      • Tombol hapus produk
      • Kolom pencarian produk
    2. Toko
      • Pada bagian ini (tab toko) Anda dapat memperbarui data toko sesuai kolom yang tersedia seperti berikut.
      • Nama Toko: Masukkan atau perbarui nama toko.
      • WhatsApp Toko: Masukkan atau perbarui nomor WhatsApp toko.
      • Alamat Toko: Masukkan atau perbarui alamat toko.
      • Catatan Toko: Masukkan atau perbarui catatan toko.
    3. Akun
      • Pada bagian ini (tab akun) Anda dapat mengelola akun Anda seperti memperbarui beberapa data berikut.
      • Nama
      • Username
      • Password
    4. Kontak Admin
      • Pada bagian ini (tab kontak admin) Anda dapat menghubungi atau mengirim pesan ke admin melalui form yang tersedia.
      • Email: Untuk mengirim pesan email ke admin melalui form email yang tersedia.
      • WhatsApp: Untuk mengirim pesan WhatsApp ke admin melalui form yang tersedia.
  • Kasir
    1. Form transaksi
      • Input transaksi manual via form dengan memasukkan nama/kode produk pada bagian kolom nama/kode.
      • Nama
      • Kode
      • Harga
      • Jumlah
      • Diskon
    2. Scan barcode
      • Input transaksi secara otomatis hanya dengan fitur scan barcode. Scan kode produk otomatis akan masuk ke tabel transaksi.
    3. Pembayaran
      • Anda dapat memproses pembayaran untuk setiap transaksi yang diinginkan.
    4. Cetak struk
      • Cetak struk untuk setiap transaksi yang diinginkan pada setiap kali Anda melakukan proses pembayaran.
    5. Edit transaksi
      • Anda dapat merubah data transaksi saat ini berdasarkan item, dengan cara klik tombol atau ikon edit.
    6. Hapus transaksi
      • Anda dapat menghapus data transaksi saat ini berdasarkan item, dengan cara klik tombol atau ikon hapus.
    7. Batalkan transaksi
      • Anda dapat membatalkan seluruh data transaksi saat ini berdasarkan nomor faktur/transaksi, dengan cara klik tombol {× Transaksi}.
    8. Pencarian
      • Anda dapat mencari data transaksi berdasarkan nomor faktur/transaksi, dengan cara klik ikon cari dan masukkan nomor faktur/transaksi pada kolom input.
    9. Menu navigasi
      • Penjualan
      • Pengeluaran
      • Panduan
      • Berlangganan
      • Dashboard
      • Logout

Cara Berlangganan

Supaya dapat terus menikmati semua fitur yang tersedia, Anda dapat berlangganan atau memperpanjang masa langganan.

Jika masa percobaan atau masa langganan telah habis, maka beberapa fitur akan dibatasi. Anda tidak dapat mengaksesnya hingga masa berlanggan diperpanjang.

Anda dapat memperpanjang masa berlangganan dengan mengikuti langkah-langkah berikut.

  • Via Notifikasi
    • Anda dapat memperpanjang masa berlangganan pada notifikasi yang muncul ketika setiap kali Anda mengakses fitur berlangganan.
    • Pada notifikasi yang muncul Anda pilih dan klik tombol "Ya, Perpanjang" untuk melanjutkan proses.
  • Via Menu
    • Anda dapa memperpanjang masa berlangganan pada bagian menu.
    • Buka bagian menu
    • Klik berlangganan
    • Klik "Ya, Perpanjang"

Cara Input Produk

Untuk proses input produk silakan Anda ikuti langkah-langkah berikut.

  1. Login
    • Silakan login dengan username dan password Anda.
  2. Menu
    • Pada halaman kasir silakan buka menu, tombol menu terdapat dibagian paling bawah.
    • Setelah menu terbuka, pilih dan klik pada pilihan "Dashboard".
  3. Dashboard
    • Pada halaman dashboard silakan Anda buka bagian tab produk, klik tombol buka.
  4. Tab Input Produk
    • Pada bagian input produk, silan Anda masukkan data produk Anda pada form input sesuai dengan masing-masing kolomnya.
      • Kode: Masukkan kode produk, atau jika produk terdapat kode barcode gunakan fitur scan barcode.
      • Nama: Masukkan detail nama produk.
      • Modal: Masukkan harga modal produk.
      • Harga: Masukkan harga jual produk.
      • Stok: Masukkan jumlah stok produk.
    • Dan pada bagian ini (tab produk) Anda juga dapat mengedit atau menghapus data produk.

Cara Input Transaksi

Untuk proses input transaksi silakan Anda ikuti langkah-langkah berikut.

  1. Login
    • Silakan login dengan username dan password Anda.

Cara Penggunaan

  1. Daftar dan buat akun di App KasirKu.
  2. Tambahkan produk dan stok barang melalui dashboard.
  3. Mulai mencatat penjualan harian menggunakan aplikasi.
  4. Akses laporan keuangan dan kelola bisnis Anda dengan mudah.
